# npm-utils
npm utils for nodejs

## install
npm install @woodw/npm-utils

## github
https://github.com/jinghewang/npm-utils

## usage
```
const utils = require('@woodw/npm-utils');
utils.test('hello boy');
```

## 参考

## publish step
 1.登录
 ```
 npm login
 ```
 2.发布
 ```
 npm publish 
 ```
 或 
 ```
 npm publish --access=public (私有域)
 ```
